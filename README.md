#  Next Step

**NextStep ou Passo a Passo** é um projeto backend desenvolvido em **Java**, criado como um desafio prático para exercitar lógica de programação, orientação a objetos e conceitos modernos de **engenharia de software**, utilizando **Spring Boot**.

A proposta do projeto é simples e poderosa: permitir que usuários criem objetivos, dividam esses objetivos em etapas e acompanhem sua evolução ao longo do tempo — tudo passo a passo.

---

##  Objetivo do Projeto

Este projeto foi desenvolvido com foco em aprendizado e prática, aplicando conceitos como:

- Lógica de programação
- Programação Orientada a Objetos (POO)
- Boas práticas de código
- Arquitetura em camadas
- Versionamento com Git
- Escalabilidade desde o início
- Desenvolvimento de APIs REST com Spring Boot

---

##  Conceito

Cada usuário pode criar **objetivos**, e cada objetivo pode ser dividido em **etapas**.

 Um objetivo **só pode ser concluído** quando todas as suas etapas forem concluídas — reforçando a ideia de progresso contínuo e estruturado.

---

##  Funcionalidades

- Cadastro de usuários
- Criação de objetivos
- Criação de etapas para cada objetivo
- Atualização do status das etapas
- Conclusão automática do objetivo ao finalizar todas as etapas
- Listagem e gerenciamento dos dados via API REST
